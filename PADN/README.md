# PADN

PADN is a dual-phase 3D CT prognostic prediction model for preoperative and postoperative CT scans. The model uses paired CT volumes, extracted clinical features, and patient-specific vascular atlas masks generated by the registration pipeline.

## Contents

- `Main.py`: model definition, dataset loading, training, validation, and testing pipeline.
- `augmentation_simple.py`: intensity-only augmentation utilities.
- `region_attention.py`: region-prior attention modules.

## Required Files Not Included

This upload folder does not include patient data, clinical CSV files, trained checkpoints, or prior-parameter CSV files.

To run the full pipeline, prepare:

- CT volumes in NIfTI format.
- Feature CSV files generated by `ClinicalPriorConstruction/main.py`.
- Patient-specific atlas files generated by `RegistrationAndSkullStripping/main.py`.
- Optional trained model checkpoints in the configured `save_dir`.
- Optional prior-parameter CSV files generated by `ClinicalPriorConstruction/calculate_region_attention_weights.py`.

Expected prior parameter location:

```text
PADN/
  prior_params/
    region_prior_params_pre.csv
    region_prior_params_post.csv
    global_prior_params.csv
```

If the regional prior CSV files are missing, `region_attention.py` falls back to neutral priors so the code can still import.

## Patient-Specific Atlas Input

The code no longer uses a single global `mni_vascular_territories.nii.gz` template. It reads individualized atlas files for each patient:

```text
RegistrationAndSkullStripping/result_v3/
  <patient_id>/
    individualized_annotation_in_preop_mni_affine.nii.gz
    individualized_annotation_in_postop_mni_affine.nii.gz
```

The atlas labels are merged into seven masks:

- ACA.
- MCA.
- PCA.
- Brainstem.
- Cerebellum.
- Cistern.
- Whole atlas support mask for IVH-related prior attention.

## Image Size

The configured input shape is:

```python
target_shape = (182, 218, 182)
```

Both CT volumes and patient-specific atlas masks are resized to this shape.

## Configuration

Edit `get_config()` near the end of `Main.py`:

```python
"save_dir": "./outputs/dl_models",
"csv_dir": "./data/csv",
"ct_data_dir": "./data/registered_ct",
"patient_template_root": "../RegistrationAndSkullStripping/result_v3",
"patient_space_prior_csv_dir": "./prior_params",
```

For external tests, update the paths in `build_external_test_sets()`.

## Run

```bash
python Main.py
```

By default, `test_only` is set to `True`, so the script expects trained checkpoints such as:

```text
best_kappa.pth
best_kappa_recall.pth
best_kappa_f1.pth
```

inside `save_dir`.
